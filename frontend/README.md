# Banking Frontend

Next.js ile geliştirilmiş web uygulaması.

## Kurulum

```bash
npm install
```

## Development

```bash
npm run dev
```

Uygulama `http://localhost:3000` adresinde çalışacak.

## Environment Variables

`.env.local` dosyası oluşturun:

```env
NEXT_PUBLIC_API_URL=http://localhost:3001
```
